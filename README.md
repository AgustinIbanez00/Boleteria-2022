# Boleteria

Es una sistema web basado en la compra online de pasajes para transporte de pasajeros.

## Aplicación Frontend:
- Blazor WebAssembly con .NET 6.

Demo: https://red-tree-004dac310.1.azurestaticapps.net/


## Aplicación Backend:
- ASP.NET Core NET6 Web Api
